# nvim
Install ``npm`` -> https://github.com/nvm-sh/nvm

Install ``neovim`` -> https://github.com/neovim/neovim

For clipboard on Linux, make sure you install ``xclip``

## Python
Make sure to install:
```bash
pip install pynvim
pip install 'python-lsp-server[all]'
```
